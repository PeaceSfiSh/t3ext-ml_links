Known problems
==============

- If you install extensions that uses the TypoScript settings "userFunc" and/or "postUserFunc" of the property "link" or
  "typolist", this extension will not work properly.


.. note::
	Please use the extension's bug tracker on Forge to report bugs: https://forge.typo3.org/projects/extension-ml_links/issues
